# ⚡ ZEUS — Zero Effort Universal System

AI-powered personal assistant with voice, weather, 
news, stocks, notes, tools and fun features!

## 🌐 Live Demo
👉 [Try ZEUS here!](https://zeus-ai-assistant-ga3bteyj5loaq4k2xduryv.streamlit.app/)

## ⚡ Features
- 🤖 AI Chat (Llama 3.3 70B via Groq)
- 🎙️ Voice Input + Output
- 🌤️ Live Weather (any city)
- 📰 Latest News
- 📈 Stock Prices
- 💱 Currency Converter
- 📝 Smart Notes
- ✅ Todo Manager
- 🔐 Password Generator
- 🌍 Language Translator
- 📊 Text Summarizer
- 😂 Jokes & Fun Zone
- ⏱️ Countdown + Pomodoro Timer
- 🎨 AI Story Generator
- 🎯 Decision Maker

## 🛠️ Tech Stack
- Python, Streamlit
- Groq API (Llama 3.3 70B)
- Open-Meteo API
- NewsAPI
- yfinance
